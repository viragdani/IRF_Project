Fontos információ: Időközben észrevettem, hogy a projekt létrehozásakor véletlenül Windows Forms App (.NET Core) template-t használhatttam, így az eddigi fejlesztésről készítek egy másolatot egy Windows Forms App (.NET Framework) template-be, és a további változtatásokat ebben folytatom.
A szoftver tervezett funkciói: Az adatok beolvasása csv fájlból történik. Adathalmaznak egy, a 2019-ben hirdetett autók adathalmazát választottam, melyet a kaggle.com-ról töltöttem le. A programom ezeket az adatokat feldolgozva lesz képes hasznos statisztikai adatokat megjeleníteni a felhasználónak (összesítő adatok, diagramok, korrelációk stb...).
A szoftver tartalmazni fog többek közt egy autó osztályt a megfelelő tulajdonságokkal, és az információk lekérdezése LINQ-val fog történni egy listából, melybe betöltöm az autókat. Az adathalmazban az autók hajtásláncáról a következő információk állnak rendelkezésre: Egy autó lehet első, hátsó, vagy négykerék-meghajtású. Ezen információk egyszerűbb kezelésére az adathalmazban módosítottam az értékeket egy azonosítóra: 1=elsőkerék-meghajtás, 2=hátsókerék-meghajtás és 3=négykerék-meghajtás. Ezt egy enumerációval fogom kezelni.
